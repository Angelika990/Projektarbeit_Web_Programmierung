## Beschreibung

Diese App besteht aus einzelnen Seiten auf denen jeweils verschiedene Favoritenlisten erstellt werden können.
Den jeweiligen Listen können neue Objekte hinzugefügt werden und neu erstelle oder bereits existierende diese könnn auch wieder gelöscht werden.
Über Knopfdruck wird der Hintergrund gewechselt.

### Technologien

- [Bootstrap v4.3](https://getbootstrap.com/)
- [Unsplash](https://unsplash.com/)
